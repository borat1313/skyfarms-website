## The Fastest Path: Vercel Drag & Drop

**1. Download** this folder: `skyfarms-site/`

**2. Go to** [vercel.com](https://vercel.com) and sign in (use GitHub/email)

**3. Click** "New Project" → "Deploy"

**4. Drag all files** from `skyfarms-site/` into the Vercel dashboard

**5. Wait 30 seconds** — your site is live at `skyfarms-website.vercel.app`

**6. (Optional) Add your domain** in Vercel Settings → Domains

Done. No Claude, no build step, no Git complexity.

---

## If You Prefer GitHub + Auto-Deploy

1. Create repo at [github.com/new](https://github.com/new)
2. Upload files via drag-and-drop on GitHub
3. Go to [vercel.com](https://vercel.com), click "New Project", select your GitHub repo
4. Vercel auto-deploys every time you push

See **DEPLOYMENT.md** for detailed steps.
